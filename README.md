# Mentis Brand Assets

Public asset host for Mentis Digital brand kit v2 (2026-04-17). These files are referenced by Canva via raw.githubusercontent.com URLs when uploading to the Mentis Brand Kit.

## Structure
- `icons/` — 24 Lucide-derived icons, recoloured to #1A3A5C
- `illustrations/` — 10 editorial vector illustrations (Recraft, palette-corrected)
- `backgrounds/` — 12 hand-coded SVG backgrounds
- `photography/` — 4 warm editorial 4K photographs (Nano Banana Pro)
- `flat-lays/` — 4 flat-lay compositions (business card, folder, letterhead, pitch book)

Brand system: see [mentis-brand-build](../mentis-brand-build) repo.
